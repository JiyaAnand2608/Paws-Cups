# 🐾 Paws & Cups - Pet Café Website

A cozy, interactive pet café website showcasing a warm and welcoming online experience for a café dedicated to pet lovers. Explore our menu, learn our story, and connect with us!

## ✨ Features
- **Responsive Navigation** - Smooth navbar with mobile menu support
- **Shopping Cart** - Add items and track totals in real-time
- **Multi-page Layout** - Home, About, Menu, Playzone, and Contact pages
- **Interactive Playzone** - Engaging activities for visitors
- **Modern Design** - Pet-themed aesthetic with welcoming visuals
- **Mobile Friendly** - Optimized for all device sizes
- **Smooth Interactions** - JavaScript-powered dynamic features

## 🛠️ Tech Stack
- **HTML** - Semantic structure (33%)
- **CSS** - Styling and layouts (42%)
- **JavaScript** - Interactive functionality (25%)

## 🚀 Installation

No installation required! This is a static website that runs directly in the browser.

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- Git (optional, for cloning)

### Steps
1. Clone or download the repository
2. Navigate to the project folder
3. Open `index2.html` in your web browser

## 💻 Usage

### Home Page (`index2.html`)
